## 仓库描述 (About)

```
基于深度学习的翻译质量检查工具，支持自动句子对齐、语义相似度计算、翻译异常检测。使用 LaBSE + Bertalign，支持 100+ 种语言。
```

英文版本：
```
AI-powered translation QA tool with automatic sentence alignment, semantic similarity calculation, and translation anomaly detection. Supports 100+ languages using LaBSE + Bertalign.
```
